## 🏠 ¿De qué trata este proyecto?

Este script de JavaScript muestra en la consola una **ficha de presentación** de un proyecto inmobiliario dentro de una plataforma de **crowdfunding inmobiliario**.

La plataforma permite que muchas personas inviertan pequeñas cantidades de dinero para financiar proyectos de construcción juntos. ¡Como construir entre todos!

## 🧮 ¿Qué calcula el script?

El script calcula automáticamente el **dinero total recaudado** en la campaña:

```
2000 inversionistas × $500.000 de aporte = $1.000.000.000
```

Esto supera la meta de **$800.000.000**, lo que significa que:
- ✅ La campaña está **cerrada** (`false`)
- ✅ La meta fue **alcanzada y superada** (`true`)

## 📚 ¿Qué aprendí esta semana?

Estos son los conceptos de JavaScript que apliqué en el proyecto:

### 🔤 String (texto)
Valores escritos entre comillas. Se usan para nombres, códigos y categorías.
```js
'Torres Bambú - Fase 1'
'ICR-001'
'Apartamentos - Uso residencial'
```

### 🔢 Number (número)
Valores numéricos sin comillas. Se usan para metas, precios y cantidades.
```js
800000000   // meta de recaudo
2000        // número de inversionistas
500000      // aporte promedio
```

### ✅ Boolean (verdadero o falso)
Solo puede ser `true` o `false`. Se usa para representar estados.
```js
false    // campaña cerrada
true   // meta alcanzada
```

### ➕ Expresión aritmética
Un cálculo que JavaScript resuelve automáticamente.
```js
2000 * 500000   // resultado: 1.000.000.000
```
